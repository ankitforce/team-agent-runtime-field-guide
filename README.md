# Team Agent Runtime Field Guide

Companion artifacts for the Substack essay: "The Team Agent Is Replacing the Chatbot."

Article URL: https://ankitkumar3514.substack.com/p/team-agent-runtime

## What This Is

A compact field guide for designing AI agents that operate inside team workflows instead of sitting in isolated chat windows.

## What A Human Can Use

Use the checklist to review whether an agent has clear context, permissions, memory boundaries, budget controls, logs, verification, and human approval gates.

## What An Agent Can Do

An agent can read this repo, draft a runtime review for a proposed team agent, produce an implementation checklist, or turn a vague work request into an agent-ready task packet.
